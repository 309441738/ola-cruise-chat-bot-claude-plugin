# Cruise Product Chat Bot · Claude Code 插件

An AI chat bot designed to assist users in finding and booking cruise products right from their chat interface.

![published with craftwave](https://img.shields.io/badge/published%20with-craftwave-6366f1)

## 安装

在 Claude Code 中执行：

```
/plugin marketplace add OWNER/ola-cruise-chat-bot-claude-plugin
/plugin install ola-cruise-chat-bot@ola-cruise-chat-bot-marketplace
```

> 将 `OWNER` 替换为本仓库所属的 GitHub 账号/组织名。

安装后即可在 Claude Code 里使用技能 **Cruise Product Chat Bot**。本仓库由
[CruiseSkillBridge](https://cruiseskillbridge.com) 按 Claude Code 插件市场规范自动生成。
